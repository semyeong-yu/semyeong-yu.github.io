- API :  
  - Drive API :  
  low-level API (not conveninent)

  - Runtime API :  
  high-level API (with nvcc)  
  included with the CUDA toolkit
  - standard CUDA API functions :  
  $$\#$$ include <cuda_runtime_api.h>
