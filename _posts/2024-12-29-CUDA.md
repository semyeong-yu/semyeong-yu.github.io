## SPMD Programming

- GPGPU Programming :  
  - serial part : in CPU C code
  - parallel part : in GPU SPMD kernel code

- SPMD (Single Program Multiple Data)

```C++
// definition
void copyArray(float *A, float *B, float *C){
  unsigned idx = threadIdx;
  if (idx < 2)
    C[idx] = A[idx];
  else
    C[idx] = B[idx];
}

// call
copyArray<< 4 >>(A, B, C); // launch 4 threads
```
